<h1>Git Cheat Sheet</h1>

Initialize repo: md [repo name] then git init

Add file to staging: git add [Filename] (or . to add all modified files)

Commit to Repo: git commit -m "commit reason"

Get status: git status

Colning repo: git clone [HTTPS Link]

Configure editor: git config --global core.editor "code --wait"

Bold: `**Bold**`

Italics: `_Italic_`

Paragraph: add `>` for every line the Paragraph will be

Images: ![Optional text](Image file)

Links: `[Text for link](Link)`

Unordered Lists: use `-,*`

Ordered Lists: use `1. 2.  3.` etc. 

Headers: 

`# Header1 `

`## Header 2`

`### Header 3`

`#### Header 4`

`##### Header 5`

`###### Header 6`
